# Vue.js 
---  

## 前言  

基于 vue2.0 写的单页面应用 Vue.js 中文社区，感谢 Vue.js 中文社区提供的API。  

( PC 端请切换至 Chrome 手机模式预览)  


## 技术栈

- vue2.0
- vue-cli
- vue-router
- axios
- muse-ui

## 实现功能

- [x] 首页列表
- [x] 下拉加载
- [x] 用户信息（点击头像）
- [x] 主题内容
- [x] 登录功能
- [x] 发布主题（支持markdown格式）
- [x] 个人中心
- [x] 主题切换

## 执行项目

> 安装依赖  
> npm install  
> 启动服务  
> npm run dev  
> 运行  
> localhost:8070/home
